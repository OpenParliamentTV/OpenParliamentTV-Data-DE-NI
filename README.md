# OpenParliamentTV-Data-DE-NI
Open Parliament TV Data Repository for the Lower Saxony state parliament (Landtag Niedersachsen)

> **Status:** in development

## Pipeline

This data is generated by the `DE-NI` parliament implementation in
[OpenParliamentTV-Tools](https://github.com/OpenParliamentTV/OpenParliamentTV-Tools):
[`optv/parliaments/DE-NI`](https://github.com/OpenParliamentTV/OpenParliamentTV-Tools/blob/main/optv/parliaments/DE-NI).
